Eklentinin doğru bir şekilde çalışabilmesi için;

1- Node.JS server çalışır halde olmalıdır. SERVER>main.js dosyasını başlatabilirsiniz.
 Ayrıca statik bir sunucu IP'si alınmadığından, uygulama üzerinden IP adresleri çalıştırılan sunucunun IP adresi ve port numarası ile yazılmalıdır.

2- eklenti WebStore'da paylaşılmadığından, "APP ID" sabit değildir. 'FirebaseAuth-Console' dan eklenti ID'nizin izinler listesine eklenmesi gerekir.

3- "Google Login API" ve "Chat-gpt API" tarafımca oluşturulmuş "API KEY" 'ler ile çalışmaktadır.

4- Veri Tabanı MSSql ile yazılmıştır.
